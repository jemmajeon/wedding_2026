통화

환율 자동 변환

"위에 환율 넣으면 자동으로 전체 원화" — 헤더 input → EXCHANGE_RATE 갱신 → BUDGET 즉시 재렌더.

평가

별점 시스템 ★/5

"추천지수도 별 5개로" — RATINGS[Day|시간] 매핑, starHTML() 함수 렌더.

대체 옵션

Alt fallback

"거기 아니면 대체해서 갈만한데도" — DATA row의 alt 필드, altHTML() 박스 렌더.

지도

Leaflet + OpenStreetMap

"실제 지도 내에서 어떻게 흘러가는지(좌표들도 구글맵 기준)" — Leaflet 라이브러리 + OSM 타일 (API 키 불필요) + Day별 색상 폴리라인.

URL

Google Maps 딥링크

"url 들 다 다시 들어갈 수 있게, 클릭하면 타고 갈 수 있게" — gmaps(query) 헬퍼로 모든 장소·예약처·호텔에 GMaps 검색 URL 부여.
